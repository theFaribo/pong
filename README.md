Pong Game

Игра рассчитана на двух игроков. Каждый игрок управляет своей ракеткой. Мяч автоматически перемещается после каждого корректного действия игрока. Побеждает игрок, который первым набрал 21 очко.

Управление

Левый игрок:

A — поднять левую ракетку вверх
Z — опустить левую ракетку вниз

Правый игрок:

K — поднять правую ракетку вверх
M — опустить правую ракетку вниз

Пропуск хода:

Space

После ввода команды нужно нажать Enter.

-----------------------------------------------------
Игровая логика

Игра работает в пошаговом режиме.

Алгоритм одного шага:

1. Программа ждёт корректную команду.
2. Если команда двигает ракетку — обновляется позиция ракетки.
3. Проверяется столкновение мяча с верхней или нижней границей.
4. Проверяется столкновение мяча с левой или правой ракеткой.
5. Мяч перемещается на одну клетку.
6. Проверяется, был ли гол.
7. Если был гол:
   - увеличивается счёт нужного игрока;
   - мяч возвращается в центр поля;
   - направление мяча сбрасывается.
8. Поле заново выводится в терминал. 9. Если один из игроков набрал 21 очко — игра завершается.
